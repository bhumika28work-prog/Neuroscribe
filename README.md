NeuroScribe 🧠📄

AI-Powered Medical Report Analysis Platform

NeuroScribe is an intelligent healthcare platform that leverages Generative AI to simplify complex medical reports. Users can upload medical documents and receive AI-generated summaries, patient-friendly explanations, and Hindi translations, making healthcare information more accessible and understandable.

Key Features
*Medical Report Upload*
Upload medical reports in PDF format.
Secure storage and processing of documents.

*AI-Powered Analysis*
Generate concise medical summaries using Google Gemini AI.
Extract key findings and important health insights.

*Patient-Friendly Explanation*
Convert complex medical terminology into simple language.
Improve understanding for non-medical users.

*Data Management*
PostgreSQL database integration using Neon.
Efficient report storage and retrieval.

*Fast & Responsive*
High-performance FastAPI backend.
