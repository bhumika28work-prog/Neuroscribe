NeuroScribe 🧠📄
AI-Powered Medical Report Analysis Platform.

 Features
- Upload Medical Reports (PDF)
- AI Generated Medical Summary
- Hindi Summary
- Patient-Friendly Explanation
- Hindi Patient Explanation
- PostgreSQL Database
- FastAPI Backend
- React + Vite Frontend
- Gemini AI Integration

Tech Stack

# Frontend
- React
- TypeScript
- Vite
- Tailwind CSS
- Axios

# Backend
- FastAPI
- SQLAlchemy
- PostgreSQL (Neon)

# AI
- Google Gemini

# Deployment

- Neon
